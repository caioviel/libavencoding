Detailed Description

This class represents a input device that used the Unix Shared Memory to generate images.

Using this AVSource, the transcoding process will utilize images in a specific buffer in memory as a raw images to generate video. An application that generate image and video must put its in a shared buffer, using the DeviceInterface class.
