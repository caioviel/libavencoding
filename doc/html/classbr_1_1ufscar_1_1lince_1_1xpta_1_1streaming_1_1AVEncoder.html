Detailed Description

This class is used to define the video and audio coding parameters of some audio and video source.

An AVEnconder's instance can represent a single audio or video stream, coded with the parameters specified by its methods. Each instance is associated with a instance of AVSource. An AVEnconder can be multiplexing in a instance and streamed by a instance of a Streaming. It also can be used to generate a output file by a AVOutputfile's instance.
